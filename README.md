# HealthCare Data Analysis Project (Interactive Dashboard Creation Using MS Excel)
## Project Objective
To Create an interactive dashboard using healthcare dataset, This healthcare analysis project transformed raw patient and hospital data into meaningful insights that support better healthcare understanding and decision-making.

## Dataset Used
- <a href="https://github.com/sohanishaikh96-ctrl/HealthCare-Data-Analysis-Project/blob/main/healthcare_dataset.csv">Dataset</a>

## Process
Collected and organized healthcare data containing patient, hospital, billing, and medical information from 2020–2024.
Cleaned the dataset by correcting text formats, handling missing values, and standardizing data for accurate analysis.
Created additional columns such as age category, billing category, admission month, and length of stay to improve analysis.
Used Excel formulas, Pivot Tables, and Pivot Charts to identify trends and patterns in healthcare data.
Analyzed patient demographics, medical conditions, billing trends, insurance usage, and hospital performance.
Designed an interactive Excel dashboard using charts, slicers, and conditional formatting for better visualization and decision-making.
